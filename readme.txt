Quick start
Requirements: .Net 5 sdk. 
Yes, I know .Net 6 will be released 11/2021 and .Net 5 will be out of support shortly after but wanted to try out .Net 5.

---
Restore
---
In solution root folder
λ dotnet restore

---
Build
---
λ dotnet build

---
Test
---
λ dotnet test

---
Run
---
In src/Publish.Api
λ dotnet run

---
Run in development
---
In src/Publish.Api
λ dotnet watch run

---
Once running, open swagger ui in a browser https://localhost:5001/swagger 
and test with the "Try me out" functionality.
---
