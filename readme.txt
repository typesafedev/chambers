Requirements: .Net 5 sdk. 
Yes, I know .Net 6 will be released 11/2021 and .Net 5 will be out of support shortly after but wanted to try out .Net 5.

Restore
---
In solution root folder run:
dotnet restore

Build
---
dotnet build

Test
---
dotnet test

Run
---
In src/Publish.Api run:
dotnet run

Run in development
---
In src/Publish.Api run:

dotnet watch run
